# Getting Started

## Install

### yarn

```bash
yarn add secullum-react-native-ui
```

### npm

```bash
npm install secullum-react-native-ui
```

## Required fonts

You should have the following font families on your project:

- Lato-Regular
- Lato-Bold

You can download it from https://fonts.google.com/specimen/Lato?selection.family=Lato:400,700.

If you use expo, you can use the `Font` module to load it in your project: https://docs.expo.io/versions/latest/sdk/font#expofontloadasyncobject.

If you use vanilla react-native, you can follow this: https://medium.com/react-native-training/react-native-custom-fonts-ccc9aacf9e5e.
